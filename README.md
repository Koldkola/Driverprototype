
  # Driver prototype

  a comprehensive ridesharing app prototype that addresses all four stakeholder perspectives: riders, drivers, platform managers, and regulators. This will include booking interfaces, dashboard views, metrics, and monitoring tools
The original project is available at https://www.figma.com/design/Agwcf3ig4K8NfRIqW8Tjnx/Driver-prototype.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
